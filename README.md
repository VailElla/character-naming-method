# 角色命名：方法、工作表与案例分析

这是一组面向小说、动画企划、游戏与科幻世界观创作的角色命名资料。它不以“寻找寓意好听的名字”为终点，而是把姓名放回角色命题、身份制度、人物关系与故事变化之中。

建议先阅读案例分析，了解方法从何而来；如果只想直接为角色取名，也可以从工作表开始。

> **剧透提示**：《可塑性记忆》案例分析涉及角色寿命、人格重置及奥莉维亚／安迪相关剧情。

## 核心思路

1. 先写角色命题，再寻找名字。
2. 先确定谁有姓氏、单名或编号，再设计具体词义。
3. 一个名字重点承担一至两项叙事任务，不必解释角色的一切。
4. 不知道隐藏词源时，名字也应当自然可用。
5. 最好的第二层含义，通常在剧情结束后才完全显现。
